# 🏥 Healthcare Analytics Project

End-to-end data analysis of a real-world healthcare dataset using Python, Pandas, SQL, and data visualization libraries.

## 📌 Project Overview
This project explores a healthcare dataset to uncover patterns in patient demographics, medical conditions, billing, and hospital operations — with the goal of generating actionable business insights.

## 🔍 Analysis Performed
- Demographic EDA (age, gender, blood type distribution)
- Medical condition trends and frequency analysis
- Patient risk segmentation
- Length of stay analysis
- Billing and revenue pattern analysis
- Seasonal admission trends
- SQL window functions for advanced querying
- Correlation analysis between key variables

## 🛠️ Tools & Technologies
- **Python** — Pandas, NumPy, Matplotlib, Seaborn
- **SQL** — Window functions, CTEs, aggregations
- **Jupyter Notebook**
- **PostgreSQL**
